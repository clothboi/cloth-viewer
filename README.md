# Textyl — cloth drape viewer

Interactive WebGL cloth draped over a sphere, with live fabric-texture swatches.
Vanilla Three.js (0.160 via unpkg importmap). Single self-contained `index.html`.

## Run locally
Open `index.html` in a browser, or serve it:
```
python3 -m http.server
```

## Hosting (GitHub Pages)
Push this folder to a GitHub repo, then enable Pages on the `main` branch (root).
